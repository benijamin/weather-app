Weather Zone


Simple weather application that provides basic informations about the current weather for a user's location (Weather, Description, Temperature, Pressure, Humidity, Min. Temperature, Max. Temperature, Wind Speed, Wind Direction), then allows to search the current weather for any location in the world, as well as a weather forecast for a specific location and period.

There are also other basic information about the application and technologies used, as well about OpenWeatherMap API services, including links.
